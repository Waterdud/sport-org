## 🚀 Быстрый старт

### 1. Установка (5 минут):

```bash
# 1. Скопируйте файлы в директорию веб-сервера
cp -r sport-org /var/www/html/

# 2. Создайте базу данных
mysql -u root -p -e "CREATE DATABASE sport_events"

# 3. Импортируйте схему
mysql -u root -p sport_events < database.sql

# 4. Настройте конфигурацию
cp config.example.php config.php
nano config.php

# 5. Установите права
chmod -R 777 uploads/

# 6. Откройте в браузере
# http://localhost/sport-org
```

### 2. Тестовые аккаунты:

- **Admin**: `admin` / `123456`
- **User1**: `user1` / `123456`
- **User2**: `user2` / `123456`
