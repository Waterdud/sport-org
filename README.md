# Веб-приложение "Спортивные Игры"

Веб-приложение для организации спортивных мероприятий (футбол, волейбол, баскетбол).

## 🚀 Технологии

- **Backend:** PHP 7.4+ (чистый, без фреймворков)
- **Database:** MySQL 5.7+
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Framework:** Bootstrap 5.3
- **Иконки:** Bootstrap Icons

## 📋 Основной функционал

1. ✅ Регистрация и авторизация пользователей
2. ✅ Создание спортивных событий
3. ✅ Просмотр списка событий с фильтрами
4. ✅ Запись на событие и отмена записи
5. ✅ Профиль пользователя с рейтингом надежности
6. ✅ История участия в событиях
7. ✅ Система оценок участников
8. ✅ Добавление и просмотр мест для игр
9. ✅ Комментарии к событиям

## 🗄️ Структура базы данных

База данных содержит 7 таблиц:

1. **users** - пользователи
2. **events** - спортивные события
3. **locations** - места для игр
4. **participants** - участники событий
5. **ratings** - рейтинги пользователей
6. **notifications** - уведомления
7. **comments** - комментарии к событиям

## 🔧 Установка и настройка

### 1. Требования

- PHP 7.4 или выше
- MySQL 5.7 или выше
- Apache/Nginx с mod_rewrite
- Composer (опционально)

### 2. Установка

1. **Клонируйте или скачайте проект:**
   ```bash
   git clone https://github.com/your-username/sport-org.git
   cd sport-org
   ```

2. **Создайте базу данных:**
   - Откройте phpMyAdmin или MySQL Workbench
   - Импортируйте файл `database.sql`
   - Или выполните команду:
   ```bash
   mysql -u root -p < database.sql
   ```

3. **Настройте подключение к БД:**
   
   Откройте файл `includes/db.php` и измените настройки:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'sport_org');
   define('DB_USER', 'root');
   define('DB_PASS', 'ваш_пароль');
   ```

4. **Создайте необходимые директории:**
   ```bash
   mkdir -p uploads/avatars uploads/locations
   chmod 755 uploads uploads/avatars uploads/locations
   ```

5. **Настройте веб-сервер:**

   **Apache (.htaccess):**
   ```apache
   RewriteEngine On
   RewriteCond %{REQUEST_FILENAME} !-f
   RewriteCond %{REQUEST_FILENAME} !-d
   RewriteRule ^(.*)$ index.php [L,QSA]
   ```

   **Nginx:**
   ```nginx
   location / {
       try_files $uri $uri/ /index.php?$query_string;
   }
   ```

6. **Откройте в браузере:**
   ```
   http://localhost/sport-org/
   ```

## 🔐 Безопасность

Приложение использует следующие меры безопасности:

- ✅ **PDO с подготовленными запросами** - защита от SQL-инъекций
- ✅ **password_hash()** - хеширование паролей (bcrypt)
- ✅ **htmlspecialchars()** - защита от XSS
- ✅ **CSRF токены** - защита от CSRF атак
- ✅ **Валидация входных данных** на сервере
- ✅ **HttpOnly cookies** для сессий
- ✅ **Prepared statements** для всех запросов к БД

## 📖 Использование

### Регистрация и вход

1. Перейдите на страницу регистрации
2. Заполните форму (имя пользователя, email, пароль)
3. После регистрации вы автоматически войдёте в систему

### Создание события

1. Нажмите "Создать событие" в меню
2. Заполните данные:
   - Название события
   - Вид спорта (футбол, волейбол, баскетбол)
   - Место проведения
   - Дата и время
   - Количество участников
   - Уровень игры
   - Описание
3. Нажмите "Создать"

### Запись на событие

1. Найдите интересующее событие
2. Нажмите "Записаться"
3. Вы будете добавлены в список участников

### Оценка участников

После завершения события:
1. Перейдите в "Мои события"
2. Найдите завершённое событие
3. Оцените участников (пришёл/не пришёл)
4. Поставьте рейтинг от 1 до 10

## 🛠️ Разработка

### Создание новых страниц

1. Подключите header:
```php
<?php
require_once 'includes/header.php';
?>
```

2. Добавьте свой контент

3. Подключите footer:
```php
<?php
require_once 'includes/footer.php';
?>
```

### Работа с базой данных

Используйте вспомогательные функции из `includes/db.php`:

```php
// Получить одну запись
$user = fetchOne($pdo, "SELECT * FROM users WHERE id = ?", [$userId]);

// Получить все записи
$events = fetchAll($pdo, "SELECT * FROM events WHERE status = ?", ['Открыто']);

// Вставить запись
$newId = insert($pdo, "INSERT INTO users (username, email) VALUES (?, ?)", [$username, $email]);

// Обновить/удалить
$affected = execute($pdo, "UPDATE users SET rating = ? WHERE id = ?", [9.5, $userId]);
```

## 🐛 Отладка

Для включения отображения ошибок добавьте в начало файла:

```php
error_reporting(E_ALL);
ini_set('display_errors', 1);
```

**Важно:** Отключите это на production-сервере!
