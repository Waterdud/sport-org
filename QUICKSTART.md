# ⚡ Быстрый старт

## 5 шагов до запуска

### 1️⃣ Импорт базы данных
```bash
mysql -u root -p < database.sql
```

### 2️⃣ Настройка подключения
Откройте `includes/db.php` и укажите свои данные:
```php
define('DB_USER', 'root');
define('DB_PASS', 'ваш_пароль');
```

### 3️⃣ Права на папку uploads
```bash
chmod 775 uploads/
```

### 4️⃣ Откройте в браузере
```
http://localhost/sport-org/
```

### 5️⃣ Зарегистрируйтесь
Создайте первого пользователя и начните использовать!

## 🔧 Основные настройки

### MySQL
- База данных: `sport_org`
- Кодировка: `utf8mb4_unicode_ci`
- Пользователь: `root` (или свой)

### PHP
- Версия: 7.4+
- Extensions: PDO, pdo_mysql
- upload_max_filesize: 10M

### Apache
- mod_rewrite: включён
- .htaccess: уже настроен

## 📝 Частые вопросы

**Q: Ошибка подключения к БД?**  
A: Проверьте настройки в `includes/db.php` и убедитесь, что MySQL запущен.

**Q: Не загружаются стили?**  
A: Проверьте, что файл `css/style.css` существует и доступен.

**Q: Не работает загрузка файлов?**  
A: Установите права `chmod 775 uploads/` и проверьте php.ini.

**Q: 500 ошибка?**  
A: Включите отображение ошибок PHP или проверьте логи сервера.
